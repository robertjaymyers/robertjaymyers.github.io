Welcome

This website features apps I've created. Some were created for practice or showcase purposes. Others were created with an official release and specific usage goals in mind. Apps marked as showcase have a downloadable and usable version, but come with an implied warning that they may be incomplete or primitive in their available features and don't have any plans for being supported if they break, or for being added to in the future.

Beyond the apps shown here, you can find some other miscellanous apps I've created on Github, as well as source code for apps that are shared here.

Cute Little Notes

Flexible and simple text editor, built for keeping track with straightforward lists and notes in the style that works best for you.

Cute Little Notes Theme Creator

Companion theme creation tool to the text editor Cute Little Notes.

Photon Match

Memory match game for language learning. Can use custom word pairs, in addition to default word pairs.

Photo Viewport

Simple photo viewer, with zooming and slideshow-like viewing capabilities.

Character Creator 2D (Showcase)

Prototype 2D Character Creation tool, with functionality for Saving, Loading, and Exporting combinations of character parts.

Tiling Puzzle Game (Showcase)

Tile puzzle game, can take any 400x400 images as puzzles.
