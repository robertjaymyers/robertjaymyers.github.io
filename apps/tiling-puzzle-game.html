Tiling Puzzle Game

Tile puzzle game, can take any 400x400 images as puzzles.

A minimalistic tile puzzle game. Puzzles are taken in as 400x400 sized images placed in the puzzles folder. You click a piece to highlight it and then click where you want to move it to. You can click middle mouse button to show a solved version of the image as a hint for what you're trying to make the puzzle look like. When a puzzle is solved, you can move onto the next one. Once all puzzles are solved, continuing will restart at the beginning, with the order shuffled.

I made this game when I was learning how to use SDL2 library and wanted to see what kind of basic games I could make with it. Like with Cute Little Notes and the ability to make custom themes for it, there is a design philosophy here of allowing people to create their own content. You can put any 400x400 image in the puzzles folder to create your own puzzles.

How to Use

The game is played by clicking on tiles to select them and then clicking on another tile to swap places between the two tiles. The puzzle is solved when the tile placement matches the original image.

  • Middle mouse button toggles visibility of a 'hint' version of the solved puzzle image, so you can see what the solved puzzle should look like if you're lost.
  • Puzzles are pulled from 400x400 png images in the puzzles folder where the program is located.
  • When a puzzle is complete, you can press space to go to the next one or x to quit.
  • If all puzzles are solved and you continue on, the program will start over on all the loaded puzzles, with the order shuffled.
