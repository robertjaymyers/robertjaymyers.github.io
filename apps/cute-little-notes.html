Cute Little Notes

Flexible and simple text editor, built for keeping track with straightforward lists and notes in the style that works best for you.

Beyond the basics of a simple text editor, Cute Little Notes offers convenience note-taking and list-making with background colors customizable to your liking to make it easy to pick out what's important to you at a glance. You can jot in comments with a colored background; jot down list titles with a colored background and automatically larger text size; jot down a marker for a list item being complete with a colored background and text of your choosing; and jot down a marker for a list item being scrapped with a colored background and text of your choosing.

Cute Little Notes aims to help those who want the flexibility of an editor like Notepad, with the formatting and convenience of color-coding and personalization; those with organizational struggles and persistent distractions, who are looking for something they can customize to their needs with a simple and eye-catching style.

Features

In addition to normal text editing capabilities, Cute Little Notes offers some convenience options for note-taking and list-making in your own style.

  • Jot
  • Customize

Jot

Jot provides convenience options for making lists and commenting on them.

  • Comment: Inserts a comment with the text you put in, using a colored background for the text.
  • List Title: Inserts a larger-sized-text list title with the text you put in, using a colored background for the text.
  • Mark Done: Inserts text, signifying that a list item is completed.
  • Mark Scrapped: Inserts text, signifying that a list item has been scrapped.

Customize

Preferences allow you to choose between various themes; ones that come with the program and mod themes if available. You can also customize colors and text relating to Jot options, overriding the existing Jot colors and text for a particular theme. These will be saved upon exiting the program normally.

  • Override Theme -> Set Jot Overrides: Set colors or text for the various Jot options. These overrides are saved relative to the current theme.
  • Override Theme -> Disable Overrides: Any existing theme overrides that are found will be ignored, rather than used in the theme.
  • Set Theme: Set what theme to use. Note that any existing colors/text from Jot notes in a file will maintain the color they were added to the file with. They will not change when the theme is changed; only new Jot notes made will use the changed-to theme.
  • Enable Mods: Choose whether the program will look for custom themes in the Mods folder.
  • Save Preferences: Manually save preference choices made. These will be saved upon exiting the program normally.
