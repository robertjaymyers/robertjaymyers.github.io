Cute Little Notes Theme Creator

Companion theme creation tool to the text editor Cute Little Notes. This tool allows you to dynamically create unique themes for Cute Little Notes by right-clicking on elements in the interface and setting their colors and images.

Once saved and completed, themes can be loaded into Cute Little Notes by being placed in the Cute Little Notes Mods folder.

Cute Little Notes Theme Creator aims to make it straightforward and intuitive to create custom themes for Cute Little Notes and share them with others, with no knowledge of stylesheets or coding required.

Saving and Loading

Themes are saved as QSS files (Qt's version of a CSS stylesheet) and create some additional files in the same location as saved.

Themes can be loaded in the Theme Creator, for further editing, by opening QSS files. The program will look for any adjacent files relevant to the theme.

  • When a QSS file is saved, the program also saves, in the same location, a themeInfo, themeJot, and images folder containing copies of the program's specified icons.
  • Any stylesheet elements not explicitly designed to be recognized by the Theme Creator for applying and editing will be ignored when loading a theme in the Theme Creator tool. This is largely irrelevant, unless you want to edit the QSS file by hand for some reason.
  • When a QSS stylesheet for a theme is loaded in Cute Little Notes, the stylesheet is applied as a whole, so you can add additional styling to a theme by hand, if you wish. Just know that unexpected styling elements won't be recognized, saved, or loaded in the Theme Creator tool; only in Cute Little Notes, the text editor.
  • For easier organization, it's recommended that you save each theme in its own folder. For example, save DarkBlue.qss in the folder "Theme - Dark Blue". This also simplifies copying the theme over to your Mods folder when it's finished.

Theme Editing: Main

The primary theme editing is done by right-clicking in parts of the interface and choosing colors or icon images for a particular element. The following are areas that can be edited, by name, and the elements within them that can be edited. The program also comes with an image guide of what areas can be edited by right-clicking.

Icon images should all be 20x20 png.

  • textEdit: Background Color, Text Color, Border Color, Scroll Background Color
  • mainToolBar: Background Color, Border Color, Separator Color, Hover Background Color (ToolBar), Hover Border Color (ToolBar), ToolBar Icon: New, ToolBar Icon: Open, ToolBar Icon: Save, ToolBar Icon: Comment, ToolBar Icon: List Title, ToolBar Icon: Mark Done, ToolBar Icon: Mark Scrapped
  • statusBar: Background Color, Text Color
  • menuBar: Background Color, Text Color, Border Color, Hover Text Color (Menu), Hover Border Color (Menu)
  • menuDropdown: Background Color, Text Color, Border Color, Separator Color, Background Color (Menu Item), Background Color (Menu Item Checked), Hover Background Color (Menu), Hover Text Color (Menu), Checked Option Image

Theme Editing: Info

The most important part of the theme is info such as name of theme, name of who made it, and version number. These are set under the Set Theme Info menu and the program will warn you if you try to save a theme without these set. This warning can be disabled under the Set Theme Info menu, if desired.

  • Creator Name: The name of who made the theme. Ideally, this should be unique and not ridiculously long, as it will show up in the name given to the theme, when it's loaded in the list of themes in Cute Little Notes.
  • Theme Name: The name of the theme. Determines how the theme is named when it's loaded in the list of themes in Cute Little Notes.
  • Version Number: Defaults to 1, or 1.0. Currently does nothing in Cute Little Notes, but in the future, may be used to determine which of two duplicate themes to use if someone has two different versions of the same theme in their Mods folder, with higher version numbers taking priority over lower.

Theme Editing: Jot

Jot theme settings determine what kind of special color or text settings will be used for the Jot features: Comment, List Title, Mark Done, and Mark Scrapped. These are edited in the Set Theme Jot menu and the available options are as follows.

  • Comment: Set Color Bg, Set Color Text
  • List Title: Set Color Bg, Set Color Text
  • Mark Done: Set Color Bg, Set Color Text, Set Text
  • Mark Scrapped: Set Color Bg, Set Color Text, Set Text
